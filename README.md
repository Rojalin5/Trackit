# Trackit - Task Tracker API

A scalable and secure RESTful Task Management API built with Node.js, Express, and MongoDB.  
Supports user authentication, task tracking with priorities, due dates, filtering, and more â€” ideal for personal productivity apps or team-based task platforms.

---

## Features

- User registration & login (JWT-based)
- Task CRUD (Create, Read, Update, Delete)
- Set task **priority**, **status**, and **due date**
- Filter, search, and sort tasks
- Authentication & Authorization middleware
- Pagination for task listing
- Centralized error handling
- Environment-based configuration
- Tested with Postman
